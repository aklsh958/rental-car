# Іконки з Figma

Щоб додати іконки з Figma:

1. **Експортуйте іконки з Figma:**
   - Виберіть іконку в Figma
   - Натисніть "Export" (праворуч)
   - Виберіть формат **SVG**
   - Натисніть "Export [назва-іконки]"

2. **Збережіть SVG файли:**
   - Створіть папку `src/components/Icons/svg/` (якщо її немає)
   - Збережіть SVG файли туди з описовими назвами:
     - `heart.svg` - іконка серця
     - `heart-filled.svg` - заповнене серце
     - `location-pin.svg` - локація
     - `check.svg` - галочка
     - `calendar.svg` - календар
     - `car.svg` - автомобіль
     - `fuel.svg` - паливо
     - `gear.svg` - шестерня
     - `chevron-down.svg` - стрілка вниз

3. **Імпортуйте в Icons.tsx:**
   - Відкрийте `Icons.tsx`
   - Додайте імпорт: `import HeartIconSvg from './svg/heart.svg';`
   - Використовуйте в компоненті

**Альтернатива:** Якщо ви хочете вставити SVG код безпосередньо в `Icons.tsx`, просто скопіюйте код SVG з Figma і вставте в компонент.

## Структура проєкту:

```
src/
├── app/                    # Next.js App Router
│   ├── page.tsx           # Головна сторінка
│   ├── catalog/
│   │   ├── page.tsx       # Сторінка каталогу
│   │   └── [id]/
│   │       └── page.tsx   # Сторінка деталей автомобіля
│   └── layout.tsx         # Корневий layout
├── components/            # React компоненти
│   ├── CarList/          # Список машин
│   ├── CarItem/          # Окрема картка машини
│   ├── CarDetails/       # Деталі автомобіля та форма
│   ├── SearchFilters/    # Компонент фільтрів
│   ├── Header/           # Навігаційний header
│   ├── NavLink/          # Компонент навігаційного посилання
│   └── Icons/            # SVG іконки
├── store/                # Zustand store
│   └── carsStore.ts      # Глобальний стан
├── services/             # API сервіси
│   └── api.ts            # Axios запити
└── types/                # TypeScript типи
    └── index.ts          # Інтерфейси
```

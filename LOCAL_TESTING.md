# Як перевірити проєкт локально

## 1. Встановіть залежності (якщо ще не встановлено):

```bash
npm install
```

## 2. Запустіть dev сервер:

```bash
npm run dev
```

## 3. Відкрийте браузер:

Перейдіть за адресою: **http://localhost:3000**

## 4. Перевірте сторінки:

### Home сторінка (http://localhost:3000):
- ✅ Відображається header з "RentalCar" та навігацією
- ✅ Є hero секція з текстом "Find your perfect rental car"
- ✅ Кнопка "View Catalog" працює

### Catalog сторінка (http://localhost:3000/catalog):
- ✅ Відображається заголовок "Catalog"
- ✅ Є фільтри: Car brand, Price/1 hour, Car mileage/km
- ✅ Завантажуються автомобілі (перевірте консоль, якщо не завантажуються)
- ✅ Кнопка "Load More" працює (якщо є більше автомобілів)
- ✅ Кнопка "Read more" на картках працює

### Деталі автомобіля (http://localhost:3000/catalog/[id]):
- ✅ Відображається детальна інформація про автомобіль
- ✅ Форма оренди працює
- ✅ Після відправки форми з'являється нотифікація

## 5. Перевірте консоль браузера (F12):

### Вкладка Console:
- ❌ Не повинно бути помилок (червоного кольору)
- ✅ Можуть бути попередження (жовтого кольору) - це нормально
- Перевірте логи API запитів:
  - "Fetching cars with params:"
  - "API response data:"
  - "Loaded cars: X"

### Вкладка Network:
- Відкрийте Network tab
- Натисніть "Search" у фільтрах
- Знайдіть запит до `/api/cars`
- Перевірте:
  - Status: має бути 200
  - Response: що повертає сервер

## 6. Перевірте функціонал:

### Фільтрація:
1. Виберіть бренд (наприклад, "Mercedes-Benz")
2. Виберіть ціну (наприклад, "50")
3. Введіть пробіг (наприклад, From: 1000, To: 50000)
4. Натисніть "Search"
5. Перевірте, чи відображаються відфільтровані автомобілі

### Обране:
1. Натисніть на іконку серця на картці автомобіля
2. Оновіть сторінку (F5)
3. Перевірте, чи зберігся вибраний автомобіль

### Пагінація:
1. Прокрутіть до кінця списку
2. Натисніть "Load More"
3. Перевірте, чи завантажилися додаткові автомобілі

## 7. Перевірка збірки:

### Збірка для production:
```bash
npm run build
```

Якщо збірка успішна - все працює правильно!

### Запуск production версії:
```bash
npm run build
npm start
```

Потім відкрийте http://localhost:3000

## Що перевірити, якщо щось не працює:

1. **Автомобілі не завантажуються:**
   - Перевірте консоль на помилки
   - Перевірте Network tab - чи виконуються запити до API
   - Спробуйте відкрити API напряму: https://car-rental-api.goit.global/api/cars

2. **Фільтри не працюють:**
   - Перевірте консоль - які параметри відправляються
   - Перевірте Network tab - що повертає API

3. **Помилки збірки:**
   - Перевірте, чи всі залежності встановлені
   - Перевірте TypeScript помилки: `npm run build`

## Корисні команди:

```bash
# Запуск dev сервера
npm run dev

# Збірка проєкту
npm run build

# Запуск production версії
npm start

# Перевірка коду (linting)
npm run lint
```
